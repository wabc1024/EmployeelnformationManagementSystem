# 								员工管理系统需求文档

员工

1.员工注册

2.员工修改自己的用户密码

3.员工可以查看自己的工资、任务等信息



管理员

1.管理员增删查改用户信息

2.发布任务



## 数据库设计

员工表：员工id(主键)，员工名，基本工资，部门id(外键)，所属管理员id

用户表：用户id(主键)，员工id，用户名，密码，注册时间，用户类别(管理员、员工)，状态(是否删除)

部门表：部门id(主键)，部门名称，创建日期

任务表：编号(主键)，创建任务的员工id(外键)，任务内容，发布日期 ## 接口

## 1.注册

请求URL :/regist

请求方式:PUT

请求体：

```json
{
    "userName":"",//8-12位数字+字母
    "userPwd":"",//6-12位数字+字母
    "empId":"" //员工Id
    //"code":""//图片验证码
}
```

响应体：
```json
//注册成功
{
    "code":200,
    "message":"success",
    "data":
    {
        "userId":"",
        "userName":"",
        "empId":"",
        "userType":"",
	}
}

//注册失败
{
    "code":501,
    "message":"userNameUsed",
    "data":null
}
```

## 2.登录

请求URL: /login

请求方式:POST

请求体：

```json
{
    "userName":"",//8-12位数字+字母
    "userPwd":"",//6-12位数字+字母
}
```

响应体：
```json
//登录成功，前端跳转到员工页面
{
    "code": 200,
    "message": "success",
    "data": {
        "empId": 3,
        "userName": "user1",
        "userPwd": null,
        "createDate": "2025-12-11T16:00:00.000+00:00",
        "userType": "emp",
        "isDeleted": false,
        "userId": 1002
    }
}
//登录失败
{
    "code":502,
    "message":"usernameNotExist",
    "data":null
}

{
    "code":503,
    "message":"passwordError",
    "data":null
}
```

## 3.用户修改密码

请求URL: /user

请求方式: PUT

请求体：

```json
{
    "userName":"",//8-12位数字+字母
    "userPwd":"",//6-12位数字+字母
    "newUserPwd",//新密码6-12位数字+字母
}
```

响应体：

```json
//修改成功,前端跳转到登录页面重新登录
{
    "code":200,
    "message":"success",
    "data":null
}
//修改失败
{
    "code":503,
    "message":"passwordError",
    "data":null
}
```

## 4.员工查看自己的信息

请求方式: GET



响应体：

````json
    "data":null

````
